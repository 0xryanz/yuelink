### 发布亮点

- 应用内更新升级为稳定版 / 预发布双通道，支持多源 manifest 拉取与 SHA-256 校验。
- 桌面端正式补齐完整发布形态：Windows 安装包与便携包、macOS 通用 DMG、Linux AppImage。
- 发布链路与桌面安装体验一起打磨：全平台 squircle 图标、Windows 安装向导图、macOS DMG 背景与修复脚本。
- 核心架构完成一轮大整理：统一 `ClashCore` 接口、拆分 `CoreManager`、XBoard API 模块化、国际化迁移到 slang。

### 产物与说明

| 文件 | 适用平台 | 说明 |
|------|----------|------|
| `YueLink-1.0.14-android-universal.apk` | Android | 通用安装包，适合绝大多数用户直接下载安装。 |
| `YueLink-1.0.14-android-arm64-v8a.apk` | Android | 64 位 ARM 设备专用包，体积更小。 |
| `YueLink-1.0.14-android-armeabi-v7a.apk` | Android | 32 位 ARM 设备专用包。 |

| `YueLink-1.0.14-windows-amd64-portable.zip` | Windows | 免安装便携版，解压后即可运行。 |
| `YueLink-1.0.14-linux-amd64.AppImage` | Linux | x86_64 AppImage，赋予执行权限后可直接运行。 |

### 校验说明

- 每个正式产物都会附带一个同名 `.sha256` 文件，用于校验下载完整性。
- Android 用户优先下载 `android-universal.apk`；只有明确知道设备架构时，才建议改下分 ABI 包。
- Windows 默认推荐 `setup.exe`；只有不方便安装或需要绿色版时，再使用 `portable.zip`。
- iOS 包为侧载用途，不走 App Store 分发。
